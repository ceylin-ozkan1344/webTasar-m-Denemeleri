# 🌐 Web Tasarım ve Front-end Çalışmalarım

HTML, CSS ve JavaScript kullanarak geliştirdiğim ilk web projelerimi bu depoda topluyorum. Görsel tasarım ve kullanıcı deneyimi üzerine yaptığım denemeleri burada bulabilirsiniz. ✨

### 📁 Projelerim ve İçerikleri

Bu depodaki her klasör, web teknolojilerini öğrenirken odaklandığım farklı bir konuyu temsil ediyor:

1.  **🧮 [01-hesapmakinesi](./01-hesapmakinesi):** Tasarım ve yerleşim mantığını kavramak için yaptığım hesap makinesi arayüzü.
2.  **👤 [02-index](./02-index):** Temel HTML yapısı üzerine çalıştığım kişisel profil sayfası denemem.
3.  **🪪 [03-kimlikkarti](./03-kimlikkarti):** CSS efektleri ve modern kart tasarımları üzerine yoğunlaştığım görsel çalışma.
4.  **🐍 [04-yilan](./04-yilan):** JavaScript mantığı ile etkileşim kurduğum klasik yılan oyunu projesi.

---

### 🛠️ Neler Öğreniyorum?
* **HTML5:** Sayfa yapısı ve etiketler.
* **CSS3:** Renkler, yerleşimler ve animasyonlar.
* **JavaScript:** Temel oyun mantığı ve etkileşim.

*Bu depo, gelişim sürecimi takip etmek ve kodlarımı arşivlemek için sürekli güncellenmektedir.* 🚀
